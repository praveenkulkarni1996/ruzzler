Ruzzler
=========

This is a tool to play [ruzzle](https://itunes.apple.com/in/app/ruzzle/id504265646?mt=8). 

### Usage

```bash
python main.py
```
and then type 16 lowercase characters. 
'abcdefghijklmop' will correspond to the board   
```
    abcd  
    efgh  
    ijkl  
    mnop  
```
a list of results will be returned.


### Test

python tests.py
